## Кроме этого

| Функция | Положение | Запятые | Комментарий | Пример |
| --- | --- | --- | --- | --- |
| Одиночное вводное словосочетание | Начало | После | <ul><li>Можно удалить без потери смысла;</li><li>можно заменить синонимами *сверх этого*, *вдобавок*, *к тому же*;</li><li>к нему или от него нельзя задать вопрос.</li></ul> | ***Кроме этого**, сегодня дождь обещали.* |
| | Середина | Перед и после | | *Мы работаем быстро и, **кроме этого**, качественно.* |
| | Конец | Перед | | *Они были далеко от нас, **кроме этого**.* |
| В обособленном обороте | В начале оборота | По границам оборота | | *Соперник не только занял первое место, **кроме этого** <ins>задев мое самолюбие</ins>, но и получил хорошую награду.* |
| | В середине оборота | Перед и после | | *Детям, <ins>подросшим</ins>, **кроме этого**, <ins>за лето</ins>, нужно было купить еще и новую одежду.* |
| Вводное словосочетание, обособленное при помощи тире или скобок | | После | | *Но чрезмерно соленый пирог (**кроме этого**, подгоревший) есть было невозможно.* |
| После сочинительного союза (*и*, *а*, *но*, *да*) | | Перед и после | Если удаление вводного словосочетания никак не влияет на структуру и смысл предложения. | *Я четко выполняю свою работу <ins>и</ins>, **кроме этого**, все делаю вовремя.* |
| | | После | Если при удалении вводного словосочетания нарушается структура предложения теряется смысл. | *Она была умна, <ins>а</ins> **кроме этого**, и красива.* |
| После союза (*и*, *а*) | Начало | После | | *<ins>И</ins> **кроме этого**, в здешнем климате твердая пшеница не вызревает.* |
| | | | | *<ins>А</ins> **кроме этого**, я еще и документы забыл взять.* |
| Перед союзом *чтобы* или *как* | | Перед и после | | *Вы должны сообщить дату своего приезда заранее, **кроме этого**, <ins>чтобы</ins> я успел все подготовить.* |
| | | | | *Вода в ручье холодная, **кроме этого**, <ins>как</ins> лед.* |
| Перед однородными членами | Середина | Перед и после (\+ тире в начале) | | *Грудному ребенку понадобятся красивые и удобные ползунки, распашонки, чепчики, носочки, — **кроме этого**, одежда должна быть из натуральных тканей.* |
| После однородных членов | | Перед (\+ двоеточие после) | | *Мне по душе совершенно другие занятия, **кроме этого**: чтение, кулинария, садоводство.* |
| В сложном бессоюзном предложении | Середина | Перед | Связано по смыслу со следующим за ним предложением или словом. | *Ветерок обдувал нас совсем слабо, **кроме этого** <ins>периодически стихал</ins>.* |
| С вводным словом или словосочетанием | | После | | ***Кроме этого**, <ins>думаю</ins>, нам стоит взять с собой запас питьевой воды.* |
| С усилительной частицей (*ведь*, *все-таки*, *все*, *все же*, *даже*, *же*, *уж*, *неужели*, *разве*) | | После | | *<ins>Ведь</ins> **кроме этого**, ему хотелось самому научиться водить машину.* |
| Предлог \+ местоимение | | Перед | <ul><li>отвечает на вопросы *кроме кого?*, *кроме чего?*</li><li>выполняет функцию дополнения</li><li>в большинстве случаев выделяется запятыми</li></ul> | *На праздник пришли все, **кроме этого** <ins>мальчика</ins>, который вчера кашлял.* |
| | | | | ***Кроме этого** <ins>платья</ins> в горошек, мне понравился еще и тот сарафан в клеточку.* |
| Оборот с предлогом | Начало или конец | Перед | Предлог «кроме» в значении исключения. | ***Кроме этого** <ins>дома</ins>, у меня больше ничего нет.* |
| | Середина | Перед и после | Предлог *кроме* в значении дополнения. | *Все картины на выставке, **кроме этого** <ins>мрачного портрета</ins>, понравились ей.* |
| | | Не ставятся | Предлог *кроме* используется в значениях *дополнительно*, *вдобавок к чему-либо*, *помимо* | *Он обнаружил, что **кроме этого** <ins>метода</ins> есть и другие способы решить задачу.* |
| | Начало | После | Оборот с предлогом «кроме» может обособляться и в значении дополнения. | ***Кроме этого** <ins>мотоцикла</ins>, у него была еще и машина, и яхта.* |
| | | | | ***Кроме этого** <ins>стихотворения</ins>, я выучил еще одно.* |

## Только если

| Позиция | Запятая | Комментарий | Пример |
| --- | --- | --- | --- |

## Однако

В большинстве случаев достаточно разграничивать *однако* как союз (близкий к *но*) и вводное слово *однако*. Если это <ins>союз</ins>, то запятую ставим только перед ним, а если <ins>вводное слово</ins> &mdash; выделяем запятыми с двух сторон.

| Позиция | Функция | Запятые | Комментарий | Пример |
| --- | --- | --- | --- | --- |
| Начало | Союз | Не ставятся | Заменяется *но* | ***Однако** зима не заставила себя долго ждать.* |
| Начало | Междометие | После | Удивление, возмущение, недоумение | «*&mdash; **Однако,** &mdash; пробормотал он, &mdash; телячьи котлеты два двадцать пять, филе &mdash; два двадцать пять, водка &mdash; пять рублей».* |
| | | | | ***Однако,** какой мороз сегодня!* |
| | | | | ***Однако!** Нам бы сейчас такие цены!* |
| Середина | Союз | Перед | Отделяет однородные члены | *Идти по обледенелому тротуару нелегко, **однако** совсем не скучно.* |
| Середина | Союз | Перед | Запятая ставится для того, чтобы разделить две части сложного предложения | *В этом году мы поставили ёлку у дивана, **однако** кот решил, что ей будет лучше там лежать.* |
| Середина или конец | Вводное слово | Перед и после | Можно убрать, и смысл предложения не изменится | *Семейный бюджет, **однако,** радуется предстоящим праздникам значительно меньше, чем дети.* |

[*Источник*](https://nbmariel.ru/content/neskuchnyy-russkiy-odnako-gde-zh-tut-zapyatye)

| Кроме, сверх | Обычно ставятся | *В конце концов мы порешили, **помимо** шампанского, абонировать для него кресло в театре, утроить жалованье, купить ему вороных, еженедельно отправлять его за город на тройках &mdash; всё это в счет Общества.* \[А. Чехов, Единственное средство.\] |
|              |                 | *Категория эта кажется нам ее собственным, а не нашим, состояньем. **Помимо этого** <ins>состоянья</ins> всё на свете названо.* \[Б. Пастернак, Охранная грамота.\] |

## В том числе

| Позиция | Функция | Запятые | Комментарий | Пример |
| --- | --- | --- | --- | --- |
| Начало | Союз | После | Без потери смысла заменяется на *а именно* | ***В том числе**, давайте обсудим, чем может закончиться эта склока.* |
| Начало | Самостоятельные слова | Не ставятся | | ***В том числе** новобранцев оказался его дед, который уже в начале войны был призывного возраста.* |
| Середина | | Перед | | *У ребенка была сильная аллергия на цитрусовые, **в том числе** на мандарины и лимоны.* |
| Середина | | Только перед союзами *и* или *а* | | *Ей нравилась романтичная музыка, <ins>а</ins> **в том числе** старинные вальсы.* |
| | | | | *Наши сослуживцы, <ins>и</ins> **в том числе** мой непосредственный начальник любили собираться в этом кафе.* | 
| | | | | *Ирина никогда не ходила в театры, <ins>и</ins> **в том числе** в тот, что стоит в центре городка.* |
| Середина | | Не ставятся | Если оборот с *в том числе* заключен в скобки | *Я предпочитаю блюда из яиц (**в том числе** пышные омлеты) на завтрак, обед и ужин.* |
| Конец | Союз | Перед &mdash; всегда. После &mdash; нет, если есть дополнительные слова | | *Нам нравится нюхать разные цветы, фиалки и пионы, **в том числе**.* |
| | | | | *Студенты получают новые навыки в течение всего учебного года, **в том числе** <ins>после прохождения практики</ins>.* |
| | | | | *Заниматься самообразованием можно где угодно, **в том числе** <ins>дома</ins>.* |

## Потому что

Если в предложении есть союз ***потому что***, то перед ним всегда ставится запятая:

> *Он не пришел, **потому что** заболел.* \[связывает части сложноподчиненного предложения\]

Иногда запятая ставится внутри:

> *Он не пришел не **потому, что** заболел.* \[входит в состав одной из частей составного союза\]

> *Это **потому, что**, когда вагон останавливается, во всем вашем теле происходит замедление скорости (А. Толстой).*
